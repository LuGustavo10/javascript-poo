Vamos Correr!

Abra o seu DevTools

Vá até o console e digite as configurações do carro.
ex: let carro1 = new Carro("fiat", "palio", 4, "true")

Depois confirme que o carro está ligado.
ex: carro1.estaLigado = true

Feche as portas do carro.
ex: carro1.PortasFechadas = true

Agora ligue o carro.
ex: console.log(carro1.ligar())

Chrome e Edge: 'F12' no seu teclado;
Firefox: 'Ctrl' + 'Shift' + 'J' no seu teclado
Ópera: 'Ctrl' + 'Shift' + 'C' no seu teclado.
